# ファクトリ  
## ファクトリの目的  
オブジェクトの生成にかかわる知識がまとめられたオブジェクト  
複雑なオブジェクトの生成処理を一か所にまとめて分かり易くする  

## ファクトリ生成の目安  
コンストラクタ内で他のオブジェクトを生成するかどうか  
コンストラクタの引数としてオブジェクトを受け取るだけならファクトリを作る必要はない

## ファクトリの実装
クラスとして実装する場合もあれば、メソッドとして実装する場合もある  

```c#:title
public class Group
{
    public Group(GroupName name, UserId id)
    {
        Name = name;
        ReaderUserId = id;
    }
    public GroupName Name {get;}
    public UserId ReaderUserId {get;}
}

public class User
{
    private readonly UserId Id;
    ///  ファクトリメソッド
    public Group CreateGroup(GroupName name)
    {
        ///　Userクラスの情報を外に漏らさないために、Userクラスにファクトリを実装
        return new Group(name, Id);
    }
}

```
