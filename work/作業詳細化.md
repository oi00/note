| フェーズ         | 行動            | 目的                  | チェック観点                 |
| ------------ | ------------- | ------------------- | ---------------------- |
| **① 依頼理解**   | 内容を正確に聞く      | 成果物のゴールを明確にする       | 何をすれば「完了」なのか明確か        |
|              | 目的・背景を確認する    | 意図を理解し、判断基準を持つ      | なぜその仕事が必要か理解しているか      |
|              | 期限・優先度を確認する   | スケジュールと優先順位を把握する    | 締切と重要度を把握しているか         |
|              | 理解を言葉で返す      | 認識のズレを防ぐ            | 相手と同じ理解になっているか         |
|              | 不明点・前提条件を洗い出す | 進行中に止まらないようにする      | 不確定な要素を整理できているか        |
|              | 記録を残す         | 認識ズレや言った言わないを防ぐ     | メモ・チャットで内容を残しているか      |
| **② 計画**     | タスク分解         | 作業を具体化・見える化する       | 作業内容がリスト化されているか        |
|              | 目標設定          | 作業と成長の両面の目的を明確にする   | 成果目標と成長目標を定義しているか      |
|              | 作業順序・依存関係整理   | 手戻りや停滞を防ぐ           | 優先度・依存関係を把握しているか       |
|              | 所要時間見積もり      | スケジュール精度を上げる        | 各タスクの工数を見積もれているか       |
|              | 逆算スケジュール      | 無理のない進行を設計する        | 期限から逆算して段取りを組めているか     |
|              | リスク洗い出し       | 想定外の停滞を防ぐ           | 詰まりそうな箇所を特定できているか      |
|              | 関係者調整         | 他者の作業も計画に組み込む       | 依頼・確認のタイミングを明確にしているか   |
|              | 計画・目標共有       | 認識のズレを防ぎ、支援を受けやすくする | 上司・依頼者に共有して確認しているか     |
| **③ 実行**     | 計画を確認してから着手   | 目的意識を保って行動する        | 今日やること・目標を明確にしているか     |
|              | タスクを一つずつ完了させる | 抜け漏れ・ミスを防ぐ          | 並行作業しすぎていないか           |
|              | 必要資料を収集       | 作業を止めない             | 必要情報を早めに揃えているか         |
|              | 進捗を見える化       | 状況を把握・共有しやすくする      | 遅れや問題を早めに伝えているか        |
|              | 迷ったら相談        | 誤った判断を防ぐ            | 判断に迷ったまま進めていないか        |
|              | 中間成果を確認してもらう  | 認識ズレを早期に修正する        | 途中段階で方向性を確認しているか       |
|              | 目標を意識して進める    | 成長意識を持って取り組む        | 計画時の目標を意識しているか         |
| **④ 確認**     | 依頼内容との照合      | 成果物が目的を満たしているか確認    | 依頼条件・目的に合致しているか        |
|              | 品質チェック        | 誤り・抜け漏れを防ぐ          | 内容・形式の誤りがないか           |
|              | 第三者視点で確認      | 相手に伝わる成果物にする        | 初見の人が理解できるか            |
|              | 他者レビュー依頼      | 客観的な品質向上            | レビュー依頼と観点共有ができているか     |
|              | 指摘反映と再確認      | 品質を最終保証             | 指摘を反映して再確認したか          |
|              | 目標達成を軽く振り返る   | 次の改善準備をする           | 目標に対して何を得られたか把握したか     |
| **⑤ 報告（納品）** | 成果物を提出        | 正しい形式で納品            | 提出ルール・フォーマットを守っているか    |
|              | 完了報告          | 状況を正確に伝える           | 内容・結果・注意点を含めて報告しているか   |
|              | 承認・確認依頼       | 公式な完了を確定させる         | 承認や受領確認を得ているか          |
|              | 振り返り          | 経験を次に活かす            | 計画時の目標に対して何を学んだか整理できたか |
|              | フィードバック共有     | 成長サイクルを定着させる        | 上司・先輩の助言を得て次に活かしているか   |
